# 重构渲染模块

请按如下要求重构渲染模块：

0. 去除硬编码的 cube 圆柱等模型创建，优化 viewdata 相关代码结构，优化渲染逻辑，修复下半部分显示不全，无法自由放大缩小，平移时模型角度会变，缩小模型会消失，光装不自然。
1. 使用 opengl 进行渲染，occ 作为集合引擎，qml 作为界面交互。
2. 支持导入 brep stp 模型文件，支持自己创建几何，并进行渲染显示。
3. 可以拖动鼠标旋转、缩放、平移模型视图，当前实现存在问题例如平移大模型很慢，缩放有时模型会消失等等。
4. 界面新增按钮，支持模型的放大、缩小、重置视图等功能。
5. 确保渲染性能流畅，界面交互响应及时。
6. 生成的代码注释使用英文。
